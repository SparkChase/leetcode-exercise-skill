# 第 1 题：两数之和

## 题目描述

给定一个长度为 `n` 的整数数组 `nums` 和一个目标值 `target`，请你在数组中找到两个不同位置的元素，使它们的和等于 `target`。

如果存在多组答案，输出任意一组下标即可。数组下标从 `0` 开始。

题目保证有且仅有一组合法答案。

## 输入描述

第一行输入两个整数 `n` 和 `target`。  
第二行输入 `n` 个整数，表示数组 `nums`。

## 输出描述

输出两个整数 `i` 和 `j`，表示满足 `nums[i] + nums[j] = target` 的两个下标。  
两个下标之间用空格分隔，顺序任意。

## 数据范围

- `2 <= n <= 2 * 10^5`
- `-10^9 <= nums[i] <= 10^9`
- `-10^9 <= target <= 10^9`

## 示例 1

```text
输入:
4 9
2 7 11 15

输出:
0 1
```

## 示例 2

```text
输入:
5 6
3 2 4 1 5

输出:
1 2
```

## 提示

- 按 Java 笔试 ACM 模式实现
- 提交时类名使用 `Main`
- 不要写 `package`
- 使用标准输入输出
- 数据量较大，优先考虑 `BufferedReader`
