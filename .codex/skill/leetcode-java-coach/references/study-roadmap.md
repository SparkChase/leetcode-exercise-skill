# Study Roadmap

Use this default roadmap for a Java learner preparing in phases. Adjust pace based on the user's background and interview timeline.

## Phase 1: Foundation

Focus:

- arrays
- strings
- hash tables
- basic simulation

Goal:

- write clean loops
- control indexes confidently
- recognize counting and lookup problems

## Phase 2: Window and Pointers

Focus:

- two pointers
- sliding window
- prefix sum

Goal:

- identify contiguous range problems
- maintain a valid window invariant

## Phase 3: Linear Data Structures

Focus:

- linked lists
- stack
- queue
- monotonic stack basics

Goal:

- become fluent with pointer updates and order-sensitive logic

## Phase 4: Search and Trees

Focus:

- binary search
- binary trees
- BST basics
- heap / priority queue

Goal:

- distinguish traversal, search-space pruning, and top-k patterns

## Phase 5: Graph and Backtracking

Focus:

- DFS
- BFS
- grid traversal
- permutations / combinations / subsets

Goal:

- manage visited state and recursive branching correctly

## Phase 6: DP and Greedy

Focus:

- 1D DP
- 2D DP
- memoization
- interval thinking
- greedy choice proofs

Goal:

- define state clearly
- justify transitions
- compare top-down and bottom-up

## Weekly Rhythm

Recommend:

1. 4 new problems in one pattern family
2. 2 review problems from previous weeks
3. 1 timed mixed problem

After each solved problem, record:

- problem title
- pattern
- time spent
- where you got stuck
- one sentence takeaway
- review date
